Python のインストール¶

Python をはじめとしたプログラミング言語というのは大雑把にいって「こう書いたら, こう動く」というルールの集合です. ルールに則ったプログラム を実行させるためのアプリケーションを「実装」とか言います. 数値計算目的で使用する場合は, PyPy というPython の実装を使うのがスタンダードにな っています.

Python言語自体には最小限のルールだけが定められています. 多くの場合で, サードパーティー製のライブラリを読み込む必要があります.このレクチャー ノートでは, Anaconda をインストールしているものと仮定しています. PyPy実装にパッ ケージ管理システムと科学技術計算に必要なライブラリを組み込んだものです. 足りないものを都度インストールするという必要がほとんどなく, 仮にあっ たとしても簡単に追加できます.

なお, Python のバージョンは 2.7 と 3.4 から選べますが, 3.4 をインストールしてください.

Windows¶

詳しくは説明しませんが, Continuum.io のウェブサイト をよく読んでインストーラを ダウンロードしてインストールしてください.

Mac OS X¶

Homebrew をインストールした上で, pyenv を使ってインストー ルすることをオススメします.

Linux¶

pyenv の使用をオススメします.
